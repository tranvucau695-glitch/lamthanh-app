# 🚀 HƯỚNG DẪN DEPLOY LAMTHANH LÊN RENDER.COM
# ================================================
# Thực hiện từng bước, mỗi bước chỉ mất 2-5 phút

## BƯỚC 1: Tạo tài khoản MongoDB Atlas (database miễn phí)
================================================
1. Truy cập: https://www.mongodb.com/atlas
2. Nhấn "Try Free" → Đăng ký bằng Google hoặc Email
3. Chọn plan: **FREE (M0)** → Next
4. Provider: AWS, Region: Singapore (ap-southeast-1) → Create Cluster
5. Đợi 1-2 phút để tạo cluster

### Tạo user database:
6. Menu trái: Security → Database Access → Add New Database User
7. Username: `lamthanh`
8. Password: Nhấn "Autogenerate" → Copy lại mật khẩu này ⚠️
9. Database User Privileges: Read and write to any database
10. Nhấn "Add User"

### Cấu hình IP whitelist:
11. Menu trái: Security → Network Access → Add IP Address
12. Chọn "Allow Access from Anywhere" (0.0.0.0/0) → Confirm

### Lấy connection string:
13. Menu trái: Database → Connect → Drivers
14. Driver: Node.js, Version: 4.1 or later
15. Copy connection string, dạng:
    `mongodb+srv://lamthanh:<password>@cluster0.xxxxx.mongodb.net/`
16. Thay <password> bằng mật khẩu đã copy ở bước 8
17. Thêm tên database: `lamthanh` sau dấu /
    Kết quả: `mongodb+srv://lamthanh:matkhau123@cluster0.xxxxx.mongodb.net/lamthanh`


## BƯỚC 2: Upload code lên GitHub
================================================
1. Truy cập: https://github.com → Đăng ký nếu chưa có
2. Nhấn "+" → "New repository"
3. Repository name: `lamthanh-app`
4. Chọn: Public
5. Nhấn "Create repository"
6. Trên trang repo vừa tạo, nhấn "uploading an existing file"
7. Kéo thả TẤT CẢ file trong thư mục `lamthanh-server` vào
   (server.js, package.json, .env.example, thư mục public/)
8. Scroll xuống → Nhấn "Commit changes"


## BƯỚC 3: Deploy lên Render.com
================================================
1. Truy cập: https://render.com → Đăng ký (dùng GitHub account)
2. Nhấn "New +" → "Web Service"
3. Connect GitHub → Chọn repo `lamthanh-app`
4. Điền thông tin:
   - Name: `lamthanh-app`
   - Region: Singapore
   - Branch: main
   - Build Command: `npm install`
   - Start Command: `node server.js`
   - Instance Type: **Free**
5. Nhấn "Advanced" → "Add Environment Variable":
   - Key: `MONGODB_URI`
   - Value: (paste connection string từ bước 1)
   - Key: `JWT_SECRET`  
   - Value: `lamthanh2025secretkey`
6. Nhấn "Create Web Service"
7. Đợi 3-5 phút deploy xong


## BƯỚC 4: Truy cập phần mềm
================================================
- Render sẽ cấp URL dạng: `https://lamthanh-app.onrender.com`
- Mở URL này trên máy tính, điện thoại, tablet đều được!
- Dữ liệu đồng bộ thời gian thực giữa tất cả thiết bị ✅
